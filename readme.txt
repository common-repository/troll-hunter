== Description ==

A plugin to automatically moderate messages and block those that contain hateful, degrading, or slanderous language. Can be configured for six languages: Swedish. English, Finnish, Danish, Russian and German. To activate the plugin, please go to its settings page and enter an API key for the Gavagai API.

== Installation ==

1. Install the plugin from the wordpress plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings page of the plugin and enter an API key from Gavagai. You can get one for free by following the link supplied in the settings page.
